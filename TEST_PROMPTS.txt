# How to use this file
#
# Paste prompts into a fresh codex chat one at a time. Each line is a real
# task; the agent's response (and any commands it runs) is the test.
#
# Run against a LIVE Vast.ai account with real credentials. Do NOT verify
# by running `vastai <cmd> --help` — that just re-confirms the source the
# skill was already audited against, so it tells us nothing. The whole point
# is whether the command actually executes against the live server and
# returns useful output. (The recent `vastai execute` discovery — accepts
# the syntax per --help, then crashes on real input — is exactly why.)
#
# Some prompts will spend money or mutate the account: launching instances,
# placing bids, creating teams/subaccounts, inviting members, running
# benchmarks. That is fine. Just clean up after (destroy instances, delete
# the test team, revoke the test API key, etc.). Budget a few dollars.
#
# Placeholders — substitute REAL values before pasting; never use fake
# numbers ("12345" / "abc123" / "endpoint 4567"), because the prompt then
# fails for an unrelated reason and tells you nothing about the skill.
#
# <OFFER_ID>          → real offer from `vastai search offers ... --raw`
# <INSTANCE_ID>       → real instance from `vastai show instances-v1 --raw -a`
# <SPOT_INSTANCE_ID>  → real spot instance (launched with --bid_price). If
#                       you don't have one, run the spot-launch prompt first.
# <TEMPLATE_HASH>     → real hash from `vastai search templates ... --raw`
# <ENDPOINT_ID>       → real endpoint id from `vastai show endpoints --raw`
#                       (these require admin-scope keys; skip the prompt if
#                       no real value is reachable)
